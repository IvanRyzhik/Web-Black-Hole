## [0.1.0] - 2026-06-18

### Added

- Initial Chrome extension scaffold (Manifest V3)
- Game overlay with hole movement, Matter.js attraction, and consumption
- Decomposer registry: buttons → layers, text → pyramids, images → tiles, empty skip
- 10-level linear growth curve
- Popup audio controls and HUD opacity slider + Exit
- Vitest unit and integration tests (18 tests)
- GitHub Actions CI
- Published to https://github.com/IvanRyzhik/Web-Black-Hole
